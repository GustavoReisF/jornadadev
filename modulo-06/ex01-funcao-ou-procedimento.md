# Exercício 01 — Função ou Procedimento?

## 1. Str()
**Resposta:** Função.

**Justificativa:** Retorna uma string convertida a partir de um número.

---

## 2. QOut()
**Resposta:** Procedimento.

**Justificativa:** Apenas exibe um texto na tela, sem retornar nenhum valor.

---

## 3. Date()
**Resposta:** Função.

**Justificativa:** Retorna a data atual do sistema.

---

## 4. Len()
**Resposta:** Função.

**Justificativa:** Retorna o tamanho de uma string ou de um array.

---

## 5. Upper()
**Resposta:** Função.

**Justificativa:** Retorna a string convertida para letras maiúsculas.

---

## 6. ClearScreen()
**Resposta:** Procedimento.

**Justificativa:** Apenas limpa a tela, sem retornar nenhum valor.
